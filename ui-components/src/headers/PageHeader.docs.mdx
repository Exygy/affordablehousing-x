## Theming Variables

<Canvas>
  <Story id="headers-page-header--style-overrides" />
</Canvas>

You can apply CSS variables to the `.page-header` class to customize the appearance of the component.

| Name                 | Type  | Description                                               |
| -------------------- | ----- | --------------------------------------------------------- |
| `--background-color` | color | The background of the header                              |
| `--border-color`     | color | The color of the top border                               |
| `--text-color`       | color | The color of text inside the header                       |
| `--title-font-size`  | size  | The font size of the title (only on medium+ size screens) |
