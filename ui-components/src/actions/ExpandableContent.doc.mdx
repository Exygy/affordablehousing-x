## Theming Variables

<Canvas>
  <Story id="headers-page-header--style-overrides" />
</Canvas>

You can apply CSS variables to the `.expandable-content` selector to customize the appearance of the component.

| Name                         | Type  | Description                                               | Default                         |
| ---------------------------- | ----- | --------------------------------------------------------- | ------------------------------- |
| `--expandable-content-container-margin-top`         | Spacing | The space between the button and the content                             | `--bloom-s6` |
| `--expandable-content-button-margin`             | Spacing | The margin around the button                               | `0`        |
