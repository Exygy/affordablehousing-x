# Expandable Content

The Expandable Content component displays a button and when clicked, toggles an arbitrary child element above.

## Component Properties

<ArgsTable of={ExpandableContent} />

You can apply CSS variables to the `.expandable-content` selector to customize the appearance of the component.

| Name                         | Type  | Description                                               | Default                         |
| ---------------------------- | ----- | --------------------------------------------------------- | ------------------------------- |
| `--expandable-content-button-margin`             | Spacing | The margin around the button                               | `0`        |
